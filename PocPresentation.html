What is Live Class Analyzer?

A system to check everyone's understanding in online classes in real-time.

1. 仕組みのイメージ

講師
映像＋音声
→ SFU →
生徒ブラウザ ×10

WebRTC＋SFUで、講師の画面と音声を一度に送信

同時配信の仕組み：
  • 講師 → SFU：一度だけ映像・音声を送信
  • SFU → 生徒：全員へ中継配信
  • 講師の負荷を低減し、多人数配信を実現
生徒映像解析の流れ：
生徒スクリーン
キャプチャ
→
Gemini APIで
画像解析
→
解析結果
（理解度等）

AIが各生徒の画面を解析し、理解度を評価

2. データ処理フロー

① 画面/音声共有開始
② 30秒ごとに
キャプチャ取得
③ AI解析 (Gemini API)
④ DB保存 &
ダッシュボード更新

3. メリット

理解が遅れている生徒を早期発見
授業改善のヒントをデータで可視化
講師の負担軽減と効率的なフォロー

4. 技術スタック & 用語解説

  • React + TypeScript (Vite)：フロントエンド
  • Python (FastAPI)：バックエンド
  • PostgreSQL：データベース
  • WebRTC + SFU：映像/音声配信
  • Gemini API：画像/音声解析
  • GCP VM + Docker Compose：インフラ (K8s検討)
  • GitLab CI：CI/CD
